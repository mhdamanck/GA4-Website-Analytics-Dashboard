📊 GA4 Website Analytics Dashboard

Tools:
• Google Analytics 4
• Power BI
• Excel

Project Overview:
Analyzed international website traffic and user behavior to support marketing strategy and campaign planning.

Key Areas:
• User Acquisition
• Engagement Analysis
• Traffic Sources
• Geographic Insights
• Demographic Analysis
• Landing Page Performance

Key Insights:
• 74% of users were new visitors
• Homepage showed lower engagement than intent-based pages
• California, Texas, and Florida emerged as high-potential markets
• Female users demonstrated higher engagement
• Mobile social traffic indicated optimization opportunities

Business Impact:
Provided data-driven recommendations for campaign planning, audience targeting, and website optimization.
